form 为了收集用户信息，让用户输入选择一些的信息。

required 代表这个是必填的，minlength、maxlength 限制输入的长度

input 有不同的状态，一开始是 default。当用户点击输入框或者使用 tab 来切换输入框，会变成 focus。

disable state 代表禁止输入

readonly state 只能读,不能修改。readonly 可以被 focus, disable 不能。

button 触发一些动作，默认啥也不做

触发表单重置，清空用户输入

触发表单提交

也可以用 input 来作为 button。input 是 void element,里面不能包裹内容。button 可以，可以包裹文字，图片等。
