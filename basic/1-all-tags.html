> 各种各样的标签

首先我们想一下网站的最基本功能是什么（可以看一下 web 的历史），显示文章。文章内容由标题、段落、图片组成，将文章的这些不同部分，命名成不同的标签，例如一级标题用 h1 表示，段落用 p 表示，图片用 img 表示。不同的标签不一样的显示效果，不一样的功能。

标签的可定制性，用不同的 attribute 来接收设置不一样的值，attribute 相当于参数。例如 input 标签用 type 属性来改变类型。img 标签用 src 属性来改变要显示的图片内容。
