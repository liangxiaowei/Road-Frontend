target 属性

> 不设置 target，默认为 "_self" 当前窗口打开

Visit freeCodeCamp(current window)

> target="_self" 当前窗口打开

Visit freeCodeCamp

> target="_blank" 新窗口打开

Visit freeCodeCamp

> target="_parent" 父窗口打开，例如嵌入了一个 iFrame, iFrame 里的一个链接为 _prarent，则会刷新外面的当前窗口

> target="_top" 最外面窗口打开，例如嵌入了两个 iFrame, 最里面的 iFrame 里的一个链接为 _top，则会刷新最外面的当前窗口

> target="_unfencedTop" 实验性功能

link state

link 有不同的状态，默认是 :link，点击过是 :visited，鼠标移上去是 :hover，获得焦点是 :focus，用户触发点击时 :active

可以用 css 来改变不同状态的样式
